# Assets / Icons

Coloca aquí los siguientes archivos antes de hacer build:

| Archivo | Tamaño | Descripción |
|---------|--------|-------------|
| `icon.png` | 1024×1024 | Ícono principal de la app |
| `adaptive-icon.png` | 1024×1024 | Ícono adaptativo Android |
| `splash.png` | 1242×2436 | Pantalla de splash |
| `favicon.png` | 32×32 | Favicon para web |
| `notification-icon.png` | 96×96 | Ícono de notificación Android |

## Generador recomendado

Usa https://expo.fyi/app-icon para generar todos los tamaños desde una imagen 1024×1024.

El diseño sugerido: sol dorado (#FFD700) sobre fondo negro (#0a0a0f),
en el estilo del sol grabado en la lápida de Chizhevsky.
